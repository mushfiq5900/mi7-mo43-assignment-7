Qus 1: Javascript kivabe kaj kore?

Java script akti high level programming language. Akti single-threaded language howa te javascript aksathe sudhu matro akti kaj korte pare. Etar mane holo javascript j kono akta code handle korte pare aki somoye. Jodio kichu khetre betikrom ache tobe javascript language hisebe synchronous kintu kichu khetre er nature asynchronous. Echarao javascript BOM, API, DOM agular maddhome kaj kore thake.

Qus 2: Javascript event loop ki kaj kore?

Javascript ekti synchronous ebong single-threaded language. Kintu kichu kaj k asynchronous er moto kore korte hoy. Ajonno javascript a j jinish ta use kora hoy setai event loop. Event loop er maddhome javascrip nije akti synchronous language howar por o asynchronous er mto kore kaj korte pare. Koyekta kaj aksathe kintu dhap a dhap a korar jonno muloto eta bebohar kora hoy.
